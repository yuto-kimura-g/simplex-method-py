# simplex-method-py
輪読会で読んだ文献[1] の p.33--36「単体表による単体法」を実装した．

## 開発環境
- Python: 3.10.2
- numpy: 1.22.2

## 動かし方
1. `$ git clone ...`
1. `$ python3 src/main.py`
